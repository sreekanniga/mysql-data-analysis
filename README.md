# SQL Data Analysis Using MySQL

## Project Overview
This project demonstrates SQL-based data analysis using MySQL on an e-commerce sales dataset. It covers fundamental SQL concepts used to retrieve, filter, summarize, and analyze structured data for business insights.

## Tools Used
- MySQL Workbench
- SQL
- E-Commerce Sales Dataset

## Files Included
- `sql_data_analysis.sql` – SQL queries
- `screenshots/` – Output screenshots
- `README.md` – Project documentation

## Key Insights
- Retrieved and explored sales records using SQL queries.
- Filtered data based on specific conditions.
- Sorted records to identify high-value sales.
- Calculated total and average sales using aggregate functions.
- Analyzed category-wise sales using GROUP BY and HAVING.
- Created SQL views for simplified analysis.
- Demonstrated table relationships using SQL JOIN operations.

## Learning Outcome
- Learned how to manipulate and analyze structured data using SQL.
- Gained practical experience with filtering, sorting, aggregation, joins, subqueries, and views in MySQL.
